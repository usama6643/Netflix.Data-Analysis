# Netflix.Data-Analysis
# Project Overview
This project is focused on practicing SQL Beginner and Intermediate queries using real-world datasets. The main goal of this project is to strengthen SQL fundamentals, improve data analysis thinking, and build confidence before moving to advanced SQL concepts.
# Objectives
Practice SQL using real datasets
Build strong understanding of:
SELECT, WHERE, GROUP BY, HAVING
Aggregate functions (COUNT, MIN, MAX, AVG)
CASE WHEN
Improve logical thinking for data analysis problems
Prepare for Junior Data Analyst roles
# Dataset Description

The datasets used in this project contain structured data similar to real business scenarios (e.g., content platforms, sales, or operations data).
Each dataset includes fields such as:
Categories / Types
Dates and Years
Ratings or Status fields
Country or Region information
Text-based attributes (titles, descriptions, etc.)
